TEG

TEG is a Reinforcement Learning enviroment, for training AI agents. This project started as my Bachelor thesis in Electrical Engineer but I have the intention of continue it and add more features that I hope would be usefull for Developers and Researchers.

Features

  1. Reinforcement Learning enviroment for a single AI agent.
  2. Robot Models (Currently UR5 only).

In the future I hope add more features like different Robot Models, reinforcement learning environments and tools that even can create your own reinforcement learning environments if you like.

Resources

Some of the resources that I have used to create this project are:

  • OpenAI Gym
  • Mujoco
  • Mujoco py
  • UR5

This project is in continuous development so you can find issues and bugs but If you wanna see more information about this project, you can visit my GitHub, I hope it would be usefull for you.
